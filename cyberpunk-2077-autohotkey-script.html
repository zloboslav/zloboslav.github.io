My Cyberpunk 2077 AutoHotkey scripts

Hi! :) Download is at the bottom, but please read my post! :)

In this blog post I write about my Cyberpunk 2077 scripts which automate the game's movement exploits.

The movement exploits in Cyberpunk can all be performed with a keyboard or game pad, but they are cumbersome, tiring and require practising the skill to perform consistently. That's why I decided to implement them with AutoHotkey in this script.

After a lot of trial and error I have found a good timing and order of executing the keystrokes. Then I discovered that the results are significantly better than doing it by hand. After testing a lot I have concluded that it is impossible to achieve such good results manually.

For example the "power jump - air dodge - power jump" combo is at least 30% higher with the script than manually performing it. Also executing a k-jump from standstill is so much easier 
The scripts are simple, but finding the timing took a while and that's why I'm proud of my achievement.

AutoHotkey

If you don't know what is AutoHotkey the following info is for you:

https://www.autohotkey.com

You don't really need to understand AutoHotkey to use the script - you can always just install it, load the script and run it. But don't forget to close it after you close the game, because it sometimes may trigger outside of the game which is very annoying. Be careful about that.

However AutoHotkey is not very difficult and you may need to modify some values depending on your cyberware anyways. Also by understanding it's basic scripting you can modify and make these scripts yourself, which is very fun and can be applied to almost any game! But do NOT use it for cheating online games or speed runs, because it's immoral and it will get you banned!

 

About my script

Currently I have implemented the following movement combos:

K-hop - stacks dodges in bullet time to obtain the speed and maintains auto-jump to keep momentum while the key is pressed. I'll try to update the k-hop to be more easily chain-able and bind it to a key allowing holding a standard direction key (WASD). Check for updates on this later. Currently I've made it to work while on the ground so it could be available as soon as you have "Kereznikov" and there wouldn't be any other requirements. Using the hotkey it doesn't chain as well as chaining manually and I'm not sure why yet.

Normal bunny hop - executes air dash and jump on key press. This still needs you to time your distance from the ground, but at least simplifies the previous 3 stroke combo into a single key, which for me is much easier and more endurable to keep up for a long time. What you need to do is to press the key in the air at consistent height before hitting the ground - then it does an air dash to accelerate and jumps from the ground maintaining momentum. If done correctly this is by far the fastest way to accelerate to great speeds - much faster than any k-hop method I've tested. However with this method it is more difficult to perform sharp turns. For this you need to hold a direction key while jumping in time. Check for updates later - I have a few ideas how to make it better.

High jump - executes power jump + air dodge + power jump with a single key. This has a second variant skipping the second jump and allowing for less height and alternate timing for your second jump, which I sometimes need. This is the highest jump possible, without modifying the game itself. For some unknown reason the timing of this script makes the jump at least 30% higher than possible manually and it is AMAZING! I'll try to rework it for a more comfortable way of activating while holding a direction key (WASD). Check for updates later.

Long jump aka k-jump - executes the k-hop boost + jump + air dodge + jump.  It can be changed to a powered jump before the dash for some additional height and distance - I'm still deciding on the final variants so check for updates on this. This also has a variation without the 2nd jump. This is executed from a standstill position enabling you to align yourself precisely before the jump. It cannot be done while moving though - stop and release any movement keys before executing.

I'm planning to include something with the Mantis Blades boost - check for updates later. I have yet to test it and see if I can discover more jumping boost combinations making the jumps even more amazing.

For more info and updates it is best to read my comments inside the script, which I keep updated while I write it. The text in this blog post will get updates only when I have time and if I don't forget.

 

Prerequisites

As explained in the previously linked clips, there are some prerequisites to be able to do the movement exploits. Without the required cyberware they are NOT possible. For your convenience please refer to the information below stating what cyberware is needed for which:

K-hop: Needs "Kereznikov" - it's very important to check which one you have because the higher grades REQUIRE you to modify the script timing in order to work. Please read about that in the script's comments. By default works with the cheapest and earliest one. I'll try to make it universal for all grades ion the future - check for update about that later.

Normal bunny hop: Needs "Manoeuvring System" for the dodge in the air which is required to be able to perform it.

High jump [power jump > air dodge > power jump]: Needs "Fortified Ankles" and "Manoeuvring System".

Long jump [k-hop > power jump > air dodge > power jump]: Needs "Kereznikov", "Fortified Ankles" and "Manoeuvring System". The same thing about the timing applies here too.
